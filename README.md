## Overview

This project implements a Python-based EEG analysis pipeline
for processing nociceptive laser-evoked potential (LEP) data.
The pipeline extracts cortical pain biomarkers — alpha band
oscillations and N2/P2 LEP components — from high-density
EEG recordings made during laser pain stimulation.

This work is motivated by my MSc dissertation (Kongala, 2025,
University of Glasgow) which characterised cold-selective
spinal projection neurons using patch-clamp electrophysiology.
This project asks the translational question: do those spinal
nociceptive circuits produce measurable cortical oscillatory
signatures in human EEG? This is directly the question
addressed by CNAP's research programme at Aalborg University.

## Dataset Description

The ds005284 dataset contains 64-channel EEG recorded from
26 young participants (average age 21 years, 18 female).
Each participant received 16 fixed-intensity laser stimuli
delivered at approximately 20-second intervals. Participants
kept their eyes open, focused on a fixation cross, and
reported pain ratings orally after each stimulation. 

| Parameter | Value |
|-----------|-------|
| EEG system | BioSemi ActiveTwo |
| Channels | 64 EEG + 1 Status |
| Sampling rate | 1024 Hz |
| Participants | 26 healthy adults |
| Stimuli | Nociceptive CO₂ laser to dorsum of hand |
| Format | BIDS-compliant (.bdf) |
| Total size | ~2.6 GB |

## Analysis Pipeline
Raw BDF file (OpenNeuro ds005284)
↓
Bad channel detection and removal (A2)
↓
Average reference applied
↓
Bandpass filter: 1–40 Hz
↓
Notch filter: 50 Hz (mains noise)
↓
┌────────────────────┬──────────────────────┐
│  Alpha Band        │  Laser Evoked        │
│  Analysis          │  Potential           │
│                    │                      │
│  Compute PSD       │  Event detection     │
│  Extract 8-12 Hz   │  (Status channel)    │
│  Peak alpha freq   │  Epoch: -200/+1000ms │
│                    │  Average epochs      │
│                    │  Extract N2/P2       │
└────────────────────┴──────────────────────┘
↓
Results + Figures

### Alpha Band Analysis

| Metric | Value | Reference |
|--------|-------|-----------|
| Peak alpha frequency | 8.0 Hz | ~10 Hz healthy adults |
| Alpha power | 1.4448 µV²/Hz | — |

Peak alpha at 8.0 Hz is 2 Hz below the typical healthy
adult value of ~10 Hz. Alpha slowing is consistently
reported in pain-sensitised individuals and is a core
biomarker measured by CNAP's research programme.

### Laser Evoked Potential

| Component | Latency | Amplitude | Reference range |
|-----------|---------|-----------|-----------------|
| N2 | 226.6 ms | 48.42 µV | 150–300 ms |
| P2 | 460.0 ms | 153.42 µV | 300–500 ms |
| NP complex | — | 105.00 µV | 50–150 µV |

Both N2 and P2 fall within published reference ranges
for healthy adults. The NP complex of 105 µV is the
primary LEP biomarker of nociceptive processing.

## Scientific Context and Motivation

### N2 Component (226.6 ms)
Reflects spino-thalamo-cortical transmission of nociceptive
signals. This is the same ascending pain pathway I
characterised at the spinal level in my MSc dissertation —
specifically, the cold-selective Calb1-expressing projection
neurons in lamina I that transmit to supraspinal targets.

### P2 Component (460.0 ms)
Reflects cortical salience and attentional processing of
pain. This is a primary target for TMS and tDCS modulation
in CNAP's brain neuro-adaptability research programme at
Aalborg University, Denmark.

### Alpha Oscillations (8.0 Hz)
Alpha slowing and suppression is the core EEG biomarker
of pain sensitisation. CNAP specifically investigates
whether non-invasive brain stimulation can restore alpha
frequency toward the healthy ~10 Hz range in chronic
pain patients.

## Limitations

- Current analysis: sub-001 only (1 of 26 participants)
- 26-subject group analysis loop is coded and ready to run
- Single channel (A1) used for N2/P2 — vertex (Cz) is
  standard; BioSemi channel mapping to standard positions
  requires further validation
- No ICA artefact rejection applied (included in
  dataset derivatives folder by original authors)
- Group-level comparison against published paper values
  pending full dataset download
